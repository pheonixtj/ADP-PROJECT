Machine Learning with Python Course Details

Course Overview

This course covers the fundamentals of machine learning, including supervised and unsupervised learning, data preprocessing, and model evaluation. Students will learn to build and train machine learning models using Python and popular libraries like Scikit-Learn and TensorFlow.

Course Content

  • Introduction to Machine Learning
  • Supervised Learning
  • Unsupervised Learning
  • Data Preprocessing
  • Model Evaluation
  • Introduction to TensorFlow
  • Building Neural Networks

Course Schedule

The course runs for 12 weeks, with two 2-hour classes per week:

  • Tuesdays and Thursdays, 6:00pm-8:00pm

Course Fees

The course fee is 2500. There is a 10% discount for early registration (at least 2 weeks before the start of the course).
