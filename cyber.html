Cyber Security Network Security Course Details

Course Overview

This course covers the fundamental concepts of network security, including cryptography, network protocols, and security architecture. Students will learn about various network attacks and how to defend against them. The course will also cover best practices for securing network infrastructure and managing network security risks.

Course Content

  • Introduction to Network Security
  • Cryptography and Encryption
  • Network Protocols and Security
  • Network Attacks and Defenses
  • Security Architecture and Best Practices
  • Managing Network Security Risks

Course Schedule

The course runs for 10 weeks, with one 3-hour class per week:
