Data Science Course Details

Course Overview

This course covers the fundamentals of data science, including statistical analysis, data visualization, and machine learning. Students will gain hands-on experience working with real-world datasets and tools like Python, Pandas, and Scikit-Learn.

Course Content

  • Introduction to Data Science
  • Descriptive Statistics
  • Inferential Statistics
  • Data Visualization
  • Introduction to Machine Learning
  • Supervised Learning
  • Unsupervised Learning
  • Deep Learning

Course Schedule

The course runs for 10 weeks, with two 2-hour classes per week:

  • Mondays and Wednesdays, 6:00pm-8:00pm
